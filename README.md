# Laravel Short Url 

Laravel Short Url allowing you to shorten urls.

## Installation

With composer

```
composer install
```

then run

```
php artisan migrate:fresh --seed
```

Enviroment Variable

```
FRONTEND_URL=http://localhost:3000
```

Demo Login

```
Email=admin@admin.com
Password=12345678
```

## Blade Scaffolding

With composer

```
npm install
```

then run

```
npm run dev
```

## Nice!

Laravel short url is now set up on your homepage.
