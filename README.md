# Laravel Short Url 

Laravel Short Url allowing you to shorten urls.

## Installation

With composer

```
composer install
```

then run

```
php artisan migrate:fresh --seed
```

finally, paste ``` ShortUrl::routes(); ``` at the end of ``` routes/web.php ```

## Blade Scaffolding

With composer

```
npm install
```

then run

```
npm run dev
```

## Nice!

Laravel short url is now set up on your homepage.
